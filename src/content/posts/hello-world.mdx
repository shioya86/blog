## このブログについて

技術的なことを中心に、学んだことや考えたことを書いていきます。

主に以下のトピックを扱う予定です。

- Web フロントエンド（Next.js, React, TypeScript）
- インフラ・クラウド
- 開発プロセス・ツール

## 技術スタック

このブログ自体も技術的な実験の場として活用していきます。現在の構成は以下の通りです。

- **フレームワーク**: Next.js 16 (App Router)
- **コンテンツ**: MDX（Markdown + JSX）
- **スタイリング**: Tailwind CSS v4
- **デプロイ**: Vercel

記事は GitHub リポジトリで管理しており、`main` ブランチへのマージで自動公開されます。
